## Persyaratan Server

PHP version 8.x

Seharusnya bekerja pada 7.x juga, tapi sangat disarankan untuk TIDAK menjalankan versi PHP yang lampau, karena berpotensi terdapat isu pada masalah keamanan dan kinerja aplikasi, serta beberapa fitur akan hilang. Mohon diperhatikan, plugin MPDF yang terpasang adalah untuk versi PHP >=7.x, jika Anda ingin menggunakan versi PHP lebih rendah, silahkan ganti plugin MPDF yang support dengan versi PHP yang Anda pilih.
